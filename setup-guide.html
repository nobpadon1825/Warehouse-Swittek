ขั้นตอนทั้งหมด: เชื่อม Google Sheets + ขึ้นเว็บด้วย Netlify

ส่วนที่ 1 — Google Sheets (เก็บข้อมูล)
1

สร้าง Google Sheets ใหม่

1
เปิด Google Sheets

ไปที่ sheets.google.com แล้วคลิก "+ Blank" เพื่อสร้างไฟล์ใหม่

2
ตั้งชื่อไฟล์

คลิกที่ "Untitled spreadsheet" มุมบนซ้าย แล้วพิมพ์ชื่อเช่น WarehouseDB

3
คัดลอก Spreadsheet ID

ดู URL ในแถบที่อยู่ของเบราว์เซอร์ จะเห็นประมาณนี้:

https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit

คัดลอกส่วนที่เป็น ตัวอักษรยาวๆ ระหว่าง /d/ และ /edit นั่นคือ Spreadsheet ID

💡
ไม่ต้องสร้าง sheet แยกเลย — Apps Script จะสร้างให้อัตโนมัติเมื่อข้อมูลถูกบันทึกครั้งแรก
2

สร้าง Apps Script (Backend API)

1
เปิด Apps Script Editor

ใน Google Sheets: คลิกเมนู Extensions → Apps Script

2
ลบโค้ดเดิมออกทั้งหมด

ใน Editor จะมีโค้ด function myFunction() {} อยู่ — ลบออกให้หมด

3
วางโค้ดจากไฟล์ Code.gs

เปิดไฟล์ Code.gs ที่อยู่ในโฟลเดอร์เดียวกับ warehouse.html แล้วคัดลอกโค้ดทั้งหมด วางลงใน Apps Script Editor

4
ใส่ Spreadsheet ID

บรรทัดที่ 6 ของโค้ด จะเห็น:

var SPREADSHEET_ID = 'ใส่ ID ของ Google Sheets ที่นี่';

เปลี่ยน 'ใส่ ID ของ Google Sheets ที่นี่' เป็น ID ที่คัดลอกมาจากขั้นตอนที่ 1 เช่น:

var SPREADSHEET_ID = '1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms';
5
กด Save (Ctrl+S)

บันทึกโค้ด

3

Deploy Apps Script เป็น Web App

1
คลิก "Deploy" มุมบนขวา

ใน Apps Script Editor คลิกปุ่ม Deploy → New deployment

2
ตั้งค่า Deployment
  • Select type: เลือก Web app
  • Description: พิมพ์ Warehouse API v1 (ไม่บังคับ)
  • Execute as: เลือก Me (your email)
  • Who has access: เลือก Anyone ← สำคัญมาก!
3
คลิก Deploy

Google จะขอให้ Authorize — คลิก Authorize access แล้วเลือก Google Account ของคุณ คลิก Allow

4
คัดลอก Web App URL

จะได้ URL ประมาณนี้:

https://script.google.com/macros/s/AKfycbxxxxxxxxxxxxxxxxxxxxxxxx/exec

คัดลอก URL นี้ไว้ — จะนำไปวางในไฟล์ warehouse.html ขั้นตอนต่อไป

⚠️
ถ้าจะแก้โค้ด Apps Script ในอนาคต ต้อง Deploy ใหม่ทุกครั้ง (New deployment) — URL จะเปลี่ยน ต้องอัปเดตใน warehouse.html ด้วย
4

ใส่ URL ใน warehouse.html

1
เปิดไฟล์ warehouse.html ด้วย Text Editor

ใช้ Notepad, VS Code, หรือโปรแกรมแก้ไขข้อความใดก็ได้

2
ค้นหาบรรทัดนี้ (Ctrl+F)
var SCRIPT_URL = '';
3
วาง URL ที่คัดลอกมา

เปลี่ยนเป็น:

var SCRIPT_URL = 'https://script.google.com/macros/s/AKfycb.../exec';

แล้วบันทึกไฟล์ (Ctrl+S)

✅
แค่นี้พอ! ตอนนี้ warehouse.html จะโหลดข้อมูลจาก Google Sheets ทุกครั้งที่เปิด และบันทึกข้อมูลกลับไปอัตโนมัติ
ส่วนที่ 2 — ขึ้นเว็บ (Netlify)
5

ขึ้นเว็บด้วย Netlify (ฟรี, ง่ายสุด)

1
เปิด netlify.com

ไปที่ netlify.com แล้วสมัครบัญชีฟรี (ใช้ Google Account ได้เลย)

2
สร้างโฟลเดอร์สำหรับ deploy

สร้างโฟลเดอร์ใหม่ในคอมพิวเตอร์ (เช่น ชื่อ warehouse-app) แล้วนำไฟล์ warehouse.html ไปไว้ข้างใน และเปลี่ยนชื่อเป็น index.html

3
Drag & Drop โฟลเดอร์

ใน Netlify Dashboard คลิก "Add new site" → "Deploy manually"

จะเห็นกล่องสีเทา ให้ลาก โฟลเดอร์ warehouse-app ทั้งโฟลเดอร์ไปวางในกล่องนั้น

4
รอ Deploy เสร็จ (~10 วินาที)

Netlify จะให้ URL เช่น funny-name-123.netlify.app — เปิด URL นี้บน Browser ใดก็ได้ ทุกที่ในโลก!

5
(ไม่บังคับ) เปลี่ยนชื่อ URL

ไปที่ Site settings → Change site name แล้วพิมพ์ชื่อที่ต้องการ เช่น my-warehouse จะได้ URL เป็น my-warehouse.netlify.app

🔄
อัปเดตในอนาคต: ถ้าแก้ไข warehouse.html และอยากอัปเดตเว็บ แค่ Drag & Drop โฟลเดอร์ใหม่อีกครั้ง Netlify จะ deploy เวอร์ชั่นใหม่ให้อัตโนมัติ

🎉 เสร็จสมบูรณ์!

ระบบคลังสินค้าของคุณพร้อมใช้งานบนเว็บแล้ว
ข้อมูลทั้งหมดถูกเก็บใน Google Sheets และสามารถเปิดดูได้จากทุกอุปกรณ์

https://my-warehouse.netlify.app

URL จริงจะขึ้นอยู่กับชื่อที่ตั้งใน Netlify
