🚀 คู่มือการตั้งค่าแบบขั้นตอน
ไม่เคยตั้งค่ามาก่อน? กดที่นี่ — ระบบจะนำทางทีละขั้นตอน
🔗 เชื่อมต่อ Google Sheets
ยังไม่ได้เชื่อมต่อ
ใส่ URL แล้วกด "บันทึก & เชื่อมต่อ"
📌 ได้จากขั้นตอน Deploy ใน Apps Script Editor → Deploy → New deployment
🔄 ซิงค์ข้อมูล

ดึงข้อมูลล่าสุดจาก Google Sheets มาแทนที่ข้อมูลในเครื่อง (ใช้เมื่อมีคนอื่นแก้ไขข้อมูลใน Sheets โดยตรง)

💾 สำรอง / กู้คืนข้อมูล

ส่งออกข้อมูลทั้งหมดเป็นไฟล์ JSON สำหรับสำรองข้อมูล หรือนำเข้าไฟล์เพื่อกู้คืน
